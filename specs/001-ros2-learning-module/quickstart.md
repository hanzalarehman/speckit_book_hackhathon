# Quickstart: ROS 2 Learning Module

This document provides a quick overview of how to get started with the ROS 2 Learning Module.

## Prerequisites

Before you begin, ensure you have the following installed:

- **ROS 2 Humble Hawksbill**: It is recommended to use Ubuntu 22.04. For other operating systems, please refer to the official ROS 2 documentation.
- **Python 3.9+**: Required for the `rclpy` examples.
- **A text editor**: For editing Python scripts and URDF files.

## How to Use This Module

1.  **Read the content**: Start by reading the documentation pages in order, beginning with the introduction. The content is designed to be read sequentially.

2.  **Run the examples**: As you go through the module, you will find code examples. It is highly recommended that you run these examples on your own machine to get hands-on experience.

3.  **Create your own node**: The module culminates in a guide to creating your own ROS 2 node. Follow the steps to create, build, and run your own "hello world" style ROS 2 node.

4.  **Create a URDF model**: The module also includes a guide to creating a URDF model. Follow the steps to create a simple humanoid robot model and view it in a URDF viewer.

## Directory Structure

The content for this module is located in the `my-website/docs/ros2-learning-module` directory.
